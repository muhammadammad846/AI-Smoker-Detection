## 🚀 Start the System (3 Steps)

### Step 1: Start Backend
```bash
cd backend
npm start
```
**Expected:** Server running on port 3000

### Step 2: Start Frontend
```bash
# In new terminal
npm start
```
**Expected:** Expo dev server starts

### Step 3: Open App
- Press `a` for Android emulator
- Or scan QR code with Expo Go app

## ⚠️ Note About Face Recognition

**Face recognition is NOT required!**

**What works without it:**
- ✅ Smoking detection
- ✅ Detection storage
- ✅ Manual challan creation
- ✅ All other features

**What's different:**
- ⚠️ Detections won't auto-identify students
- ✅ Admin manually selects students when creating challans
- ✅ System works perfectly this way

**To add face recognition later:**
- Install CMake and Visual C++ Build Tools
- Run: `pip install face-recognition`
- System will then auto-identify students
